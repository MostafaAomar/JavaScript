Inline-Tags

Ein Inline-Tag steht um einen Text herum und hat nur Einfluss auf die Stelle an dem der Text steht

Hier steht das was dem User als Link angezeigt wird, das kann auch ein Bild sein
a ist ein s.g. inline-Tag
weitere inline-Tags sind b, i, u, strong....
b macht etwas fett i macht etwas kursiv u unterstreicht etwas strong macht ebenfalls etwas fett

Die zuletzt genannten Tags dienen keiner Design-Auszeichnung mehr (auch wenn sie die ursprünglichen Designs beibehalten haben) sondern einem semantischen Gebrauch

CSS

Was ist CSS?

  • CSS = Cascading Style Sheets
  • CSS ist eine Beschreibungssprache
  • mit CSS werden html-Tags gestaltet
  • Nutzen: Strikte Trennung zwischen Inhalt (HTML) und Design (CSS)

es gibt drei Arten von CSS

  1. Inline CSS - wird direkt in einem HTML-Tag definiert
  2. Internal CSS - wird im head einer HTML-Seite definiert
  3. External CSS - wird in einer eigenen Seite definiert
External CSS ist "das Wichtigste", weil
  • es kann von mehreren HTML-Dokumenten referenziert werden, so dass alle auf den gleichen Style zugreifen
  • es ist leicht zu warten
  • vollständige Trennung von Inhalt und Gestaltung
